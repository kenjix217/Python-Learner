Official Python Website:

üì• Download Python 3.12

  2. ‚úÖ CRITICAL: Check "Add Python to PATH"

  3. Follow installation wizard

Linux Users:

Ubuntu/Debian:

sudo apt install python3.11 python3-pip

üì¶ Step 3: Understanding pip

pip is Python's package installer.

Install a package:

pip install requests

Install multiple packages:

pip install flask pandas matplotlib

List installed packages:

pip list

üåê Step 4: Virtual Environments

What are virtual environments?
Isolated Python environments for each project to prevent package conflicts.

Windows:

python -m venv venv
venv\Scripts\activate

Mac/Linux:

python3 -m venv venv
source venv/bin/activate

You'll see (venv) in your terminal when activated.

Deactivate:

deactivate

üÜò Troubleshooting

Problem: "python not found" after installation

Solution (Windows): Reinstall and check "Add Python to PATH"

Solution (Mac): Use python3 instead of python

Problem: "pip not found"

python -m pip --version

Use python -m pip install instead of just pip install

# What is Programming?

## Goal

By the end of this lesson, you will understand what programming is, why it is useful, and why Python is a great first language to learn.

---

## Explanation

Programming is giving instructions to a computer to perform tasks. Just like you might follow a recipe to bake a cake, a computer follows a program to complete a task.

Think of it this way: when you tell a friend how to get to your house, you give them step-by-step directions. Programming is the same idea, except you are talking to a computer instead of a person.

**Why do we program?**

Computers are very fast and very precise. They can do repetitive tasks without getting tired or making mistakes. But they are not smart on their own. They need clear, exact instructions. That is what a program is‚Äîa set of instructions written in a language the computer understands.

**Why Python?**

Python is one of the most beginner-friendly programming languages. It reads almost like plain English. You can write powerful programs with just a few lines of code. It is used everywhere: websites, data science, games, and even robots.

Here is what makes Python special:

- Easy to read and write
- Used by professionals in many fields
- A huge community ready to help beginners
- Free and works on any computer

---

## Example

Here is a very simple Python program:

```python
print("Hello, world!")
```

**What this does:**

This program displays the message "Hello, world!" on the screen.

**Let's break it down:**

- print() is a command that tells Python to show something on the screen
- The text inside the parentheses and quotes is what will be displayed
- When you run this program, Python shows:

```
Hello, world!
```

That is it. You just saw a complete Python program.
